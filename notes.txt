### CODE STYLE ###

VARIABLE NAMES
 - Class variables (everything but static final):
   	 variableName_
 - Class variables (static final, except serialVersionUID):
   	 VARIABLE_NAME
 - Local variables:
   	 variableName

DEFINITION ORDER
 - Classes:
	 (public) (final) class ClassName
 - Variables
	 (public) (static) (final) <type> <variable name>
 - Methods
	 (public) (static) (void/<type>) <method name>

JAVADOC
 - Configurations:
	 - Scope: Whole project (include test sources)
	 - Include JDK and library sources in -sourcepath
	 - Link to JDK documentation (use -link option)
	 - Output directory: chatty/target/javadoc
	 - Visibility level: private
	 - Generate hierarchy tree
	 - Generate navigation bar
	 - Generate index (separate index per letter)
	 - @use
	 - @author
	 - @deprecated
	 - Generate with mvn javadoc:javadoc
 - Always add javadoc to:
         - Classes
         - Methods
	 - Interfaces
	 - Enums
	 - Class Fields (everything but private)
